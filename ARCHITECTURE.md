## ðŸŽ¯ Key Design Decisions

1. **TanStack Query over Redux**
   - Simpler setup for API state
   - Built-in caching and refetching
   - Less boilerplate code

2. **Batch Thumbnails over Individual Requests**
   - 25 thumbnails per API call
   - Reduces network overhead
   - Better user experience

3. **Session-based Auth over JWT**
   - Simpler server-side revocation
   - Better for small-scale apps
   - HTTP-only cookies more secure

4. **Dropbox over S3**
   - Easier setup and management
   - Built-in UI for file management
   - Suitable for wedding gallery scale

5. **Lambda Components over Class Components**
   - Modern React best practices
   - Better with hooks
   - Cleaner, more readable code

6. **Custom Hooks over Inline Queries**
   - Reusable logic
   - Easier testing
   - Better organization
